# Reto Hackaton-Comercio 2025
 Este proyecto fue desarrollado como solución a un reto presentado en la hackathon de 2025 organizada por el diario el Comercio. El objetivo principal del desafío fue crear en 48 horas un aplicativo que brinde a los ciudadanos informacion utilitaria referida a los procesos electoriales que se enfrentan en el 2026.
